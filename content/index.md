Hi 我是 Jay，我的主职是前端工程师，同时也是位独立开发者。

近期，我上线了一个全新的工具类网站：[Base64 SVG Encoder]。基于 Modern Web API 开发，功能是将 SVG 图片快速转换为 Base64 编码，以便应用到产品的开发中去。

和其它同类网站相比：

- ⚡️ 速度快，几乎瞬间输出结果。
- 🎨 样式简洁美观，参考 [Lucent UI] 开发。
- ⚙️ 支持多图上传、一键复制。
- 🔒 无后端服务，不必担心隐私泄漏。
