## 新产品

说回近期，我刚刚上线了一个全新的网站：[Base64 SVG Encoder](https://b64.imjay.dev/)。它基于 Modern Web API 开发，作用是将 SVG 图片快速转换为 Base64 编码，以便应用到产品的开发中去。

和其它同类网站相比：

- ⚡️ 它的速度很快，几乎可以瞬间输出结果。
- 🎨 样式简洁美观，对不同尺寸的设备均有不错的 UI 支持。
- ⚙️ 支持多图上传、快速复制，将效率最大化。
- 🙈 它不需要后端服务，因此您完全不必担心隐私会被泄漏。
